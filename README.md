# rcalc
simple command line utility for extremely simple RPN calculator

# installing

- clone the project
- build with cargo 

# guide

- only integers are supported
- enter the value and press RET/ENTER key for pushing it onto stack
- then press + or - or * or / this will trigger a operation that will pop 
  top two elements and pushes the result on to stack again
- enter 'q' at any point of time to exit rcalc
